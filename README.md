<h1 align="center"><span>SIMPATI</span></h1>

<p align="center"><span align="center">SIMPATI (Sistem Pendaftaran Studi Independen) adalah sebuah full-stack web application yang memanajemen pendaftaran program studi atau kursus studi independen</span><p>

## Fitur-Fitur

### Student

- Daftar akun
- Masuk akun
- Pengisian biodata
- Daftar program studi
- Pembayaran program studi
- Pengunduhan LoA
- Kontak admin
- Mengganti email dan password

### Admin

- Daftar akun
- Masuk
- Mengelola program studi
- Mengelola pembayaran program studi
- Mengelola LoA
- Kontak pengembang
- Mengganti email dan password
